# Инструкция по сборке APK

### 1. Склонируйте репозиторий:

```git clone https://github.com/NoDreams2/android_weather.git```

### 2. Перейдите в папку:

```cd android_weather```

### 3. Соберите APK:

```./gradlew assembleDebug```

### 4. APK находится в:

app/build/outputs/apk/debug/app-debug.apk
